# i3 wm (window manager) example config files
A better tiling and dynamic window manager

![i3 example](screen.png?raw=true "Desktop preview")

Required packages:

- Fontawesome

tested and best works with:

- i3, version: 4.13
- i3status, version: 2.10
- i3lock, version: 2.7
- FontAwesome 4.7
- lilyterm
